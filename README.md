## ai-sugggestion-textarea

Similar to Github Copilot, it suggests the continuation of the string entered in the text form.

This was created with Next.js.

## Getting Started

### Environment variables

```bash
# .env.local
OPENAI_API_KEY=sk-**
```

### Run the development server

```bash
npm run dev
# or
yarn dev
# or
pnpm dev
# or
bun dev
```
